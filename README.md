# Getting Started

<h3>Clonning the Repository</h3>

```
$ git clone git@github.com:fernandoors/genome.git

$ cd genome
```

<h3>Installing dependencies</h3>

```
$  yarn
```

<h3>Run Project</h3>

```
$  npx react-native run-ios
```

OR

```
$  npx react-native run-android
```
